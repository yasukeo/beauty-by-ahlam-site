# Beauty by Ahlam — Maquette de site

Maquette de démonstration pour le salon **Beauty by Ahlam** (Témara), réalisée pour montrer ce qu'un vrai site pourrait apporter au salon : visibilité Google, présentation des soins et des packs, et prise de contact en un clic.

⚠️ Ceci n'est **pas** le site officiel de l'établissement.

## Structure

```
.
├── index.html        page unique
├── css/
│   └── styles.css    tous les styles
├── js/
│   └── main.js        menu burger + animations au scroll
└── README.md
```

## Aperçu local

Aucune installation nécessaire — ouvre simplement `index.html` dans un navigateur, ou utilise un petit serveur local :

```bash
npx serve .
```

## Déploiement

Ce dossier peut être déployé tel quel sur GitHub Pages, Vercel ou Netlify (aucune étape de build requise).

## Contenu

- **Nos soins** — hammam, soin du visage, manucure & pédicure, coiffure
- **Nos packs** — formules réelles du salon (199 DH, 349 DH, 249 DH, 399 DH)
- **Le rituel** — déroulé d'une séance hammam en 4 étapes
- Avis clients, galerie photo, horaires et coordonnées

Les images de soins et de packs sont générées (Higgsfield) pour la démonstration ; les photos du salon dans le héro et la galerie sont les vraies photos Google du salon.
